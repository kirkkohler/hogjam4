# Set up

```
npm install
```

# Run Application
---

```
npm start
```

# Deploy to GitHub
---

```
npm run deploy
```

This command updates the javascript bundle, commits it, and pushes the update to gh-pages.

You can also update the javascript bundle on its own using this:

```
npm run bundle
```
